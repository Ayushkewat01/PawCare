# Sunandani Web Project

This is a Flask-based backend web application.

## Prerequisites
- Python 3

## Getting Started

Follow these steps in your terminal to get the project running locally.

**1. Activate the Virtual Environment**
Activate the isolated Python environment so you don't install dependencies globally.
```bash
source venv/bin/activate
```

**2. Install Dependencies** 
*You only need to run this command the very first time, or if you add new libraries to `requirements.txt`.*
```bash
pip install -r requirements.txt
```

**3. Run the Server**
Start the local development server.
```bash
python3 app.py
```

**4. View the App**
Open your web browser and go to:
[http://localhost:5001](http://localhost:5001)

*(To turn off the server when you are done, just go back to your terminal and press `CTRL + C`)*
