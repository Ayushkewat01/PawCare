## 📖 Overview

**PawCare** is an aesthetically pleasing, responsive web platform designed for modern pet owners. It offers a seamless experience for exploring veterinary services, booking online consultations, and accessing expert pet care resources. Built with a focus on premium UI/UX, PawCare ensures a calming and easy-to-navigate interface for both you and your beloved companions.

## ✨ Features

- **Modern & Responsive UI**: Beautifully crafted interface using sophisticated typography and a premium color palette (Sage, Terracotta, Cream).
- **Virtual Consultations**: Intuitive booking section to easily schedule online and offline appointments with specialized veterinarians.
- **Comprehensive Services**: Detailed sections covering everything from full checkups and dental care to customized nutritional plans.
- **Expert Veterinarians**: Explore profiles of vetted professionals, complete with reviews, specialties, and real-time availability indicators.
- **Smooth Animations**: Incorporates elegant CSS micro-animations, glassmorphism elements, and hover effects for an engaging user experience.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3
- **Backend / Server**: Python 3, Flask framework
- **Typography**: Google Fonts (`DM Sans` for body, `Playfair Display` for headings)

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

Ensure you have the following installed:
- Python 3.8+

### Installation & Run Instructions

**1. Clone the repository:**
```bash
git clone https://github.com/Ayushkewat01/PawCare.git
cd PawCare
```

**2. Activate the Virtual Environment:**
Activate the isolated Python environment.
*(Mac/Linux)*:
```bash
source venv/bin/activate
```
*(Windows)*:
```bash
.\venv\Scripts\activate
```

**3. Install Dependencies:**
Wait for the terminal to show `(venv)` then install the required Python packages:
```bash
pip install -r requirements.txt
```

**4. Start the Application:**
Run the Flask server:
```bash
python app.py
```

**5. View the Website:**
Open your browser and navigate to:
[http://localhost:5001](http://localhost:5001)

*(To stop the server, press `CTRL + C` in your terminal.)*

## 🎨 Design System

PawCare utilizes a curated design system to maintain visual consistency and a premium feel:
- **Primary Colors**: Sage Green (`#7A9E7E`), Dark Sage (`#4A6B4E`)
- **Accent Colors**: Terracotta (`#C97B4B`), Warm Cream (`#FDF8F2`)
- **Typography**: `Playfair Display` for elegant headings and `DM Sans` for readable, modern body text.
- **Styling Details**: Frosted glass navigation bars, CSS transitions, and distinct content sections.
